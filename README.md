### 👾 Davinci 개발 목적 👾
##### AI개발이 불러일으키는 환경문제에 관한 경각심을 일깨우고자 Davinci를 만들었다.


##### 즉, 인공지능 산업의 이면에 존재하는 문제점을 비판하고자한다. 


##### 그리고 공익성을 지니면서 지루하지 않은 게임을 만들기 위해 추리 요소를 추가했다.
